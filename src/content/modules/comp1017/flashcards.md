## Key Concept Flashcards

| # | Question | Answer |
|---|----------|--------|
| 1 | What is a set? | An unordered collection of distinct objects called elements or members. |
| 2 | What is the difference between a subset and a proper subset? | A subset (A ⊆ B) means all elements of A are in B; a proper subset (A ⊂ B) means A ⊆ B and A ≠ B. |
| 3 | What is De Morgan's Law for sets? | (A ∪ B)' = A' ∩ B' and (A ∩ B)' = A' ∪ B'. |
| 4 | What is a function (formally)? | A relation f: A → B where every element in A maps to exactly one element in B. |
| 5 | What does it mean for a function to be bijective? | It is both injective (one-to-one) and surjective (onto). |
| 6 | What is a relation on a set A? | A subset of the Cartesian product A × A. |
| 7 | What are the three properties of an equivalence relation? | Reflexive, symmetric, and transitive. |
| 8 | What is modus ponens? | If P → Q is true and P is true, then Q is true. |
| 9 | What is a contrapositive? | The contrapositive of P → Q is ¬Q → ¬P; they are logically equivalent. |
| 10 | What is a proof by contradiction? | Assume the negation of the statement, then derive a logical contradiction. |
| 11 | State the Division Algorithm. | For integers a and d > 0, there exist unique q and r such that a = dq + r where 0 ≤ r < d. |
| 12 | What is the GCD and how is it computed? | The greatest common divisor is the largest integer dividing both numbers; computed using the Euclidean algorithm. |
| 13 | What is the Fundamental Theorem of Arithmetic? | Every integer greater than 1 can be uniquely expressed as a product of primes (up to ordering). |
| 14 | What is the formula for an arithmetic sequence sum? | S_n = n/2 × (2a + (n-1)d) or S_n = n/2 × (first + last). |
| 15 | What is the formula for a geometric series sum (finite)? | S_n = a(1 - r^n) / (1 - r) for r ≠ 1. |
| 16 | What is mathematical induction? | Prove base case P(1), then prove P(k) → P(k+1) for arbitrary k; conclude P(n) for all n. |
| 17 | What is the derivative of x^n? | nx^(n-1) (the power rule). |
| 18 | What is the chain rule? | d/dx[f(g(x))] = f'(g(x)) × g'(x). |
| 19 | What is the integral of x^n (n ≠ -1)? | x^(n+1) / (n+1) + C. |
| 20 | What does it mean for a relation to be a partial order? | It is reflexive, antisymmetric, and transitive. |
| 21 | What is the power set of a set A? | The set of all subsets of A, denoted P(A), with |P(A)| = 2^|A|. |
| 22 | What is logical equivalence? | Two propositions are logically equivalent if they have the same truth value in every possible interpretation. |
| 23 | What is the difference between ∀ and ∃? | ∀ (for all) means the predicate holds for every element; ∃ (there exists) means at least one element satisfies it. |
| 24 | What is the product rule in combinatorics? | If task A has m ways and task B has n ways, then doing both has m × n ways. |
| 25 | What is the definition of a convergent sequence? | A sequence converges to L if for every ε > 0, there exists N such that for all n > N, |a_n - L| < ε. |
