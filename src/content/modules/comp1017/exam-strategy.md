## Time Allocation

- **Total exam time:** Typically 2 hours
- **Reading time:** First 5 minutes — read all questions, identify easy wins
- **Per question:** Allocate time proportional to marks (e.g., 2 min per mark)
- **Buffer:** Leave 10 minutes at the end for checking arithmetic and logic

## Question Types to Expect

- **Calculation:** Evaluate GCDs, derivatives, integrals, series sums
- **Proof:** Direct proof, contradiction, induction (very common)
- **Definition/Explanation:** State definitions, explain properties
- **Set manipulation:** Prove set identities, determine subsets
- **True/False with justification:** Identify whether statements about functions/relations hold

## Key Formulas to Memorise

1. De Morgan's Laws: (A ∪ B)' = A' ∩ B', (A ∩ B)' = A' ∪ B'
2. Arithmetic series: S_n = n/2 × (2a + (n-1)d)
3. Geometric series: S_n = a(1 - r^n)/(1 - r)
4. Euclidean algorithm for GCD
5. Power rule: d/dx(x^n) = nx^(n-1)
6. Product/Chain rule for derivatives
7. Integration by parts: ∫u dv = uv - ∫v du
8. Binomial coefficient: C(n,k) = n! / (k!(n-k)!)
9. Induction template: base case + inductive step
10. Logical equivalences (contrapositive, De Morgan's for logic)

## Night Before Checklist

1. ☐ Can you write a proof by induction from scratch?
2. ☐ Do you know all set operation laws (De Morgan, distributive, etc.)?
3. ☐ Can you apply the Euclidean algorithm quickly?
4. ☐ Are you comfortable with function properties (injective, surjective, bijective)?
5. ☐ Can you determine if a relation is an equivalence relation or partial order?
6. ☐ Do you remember derivative rules (chain, product, quotient)?
7. ☐ Can you evaluate definite integrals?
8. ☐ Do you know how to test series convergence?
9. ☐ Can you construct truth tables and simplify logical expressions?
10. ☐ Can you solve basic combinatorics problems (permutations, combinations)?
