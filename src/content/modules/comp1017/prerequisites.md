## What You Need to Know Before Starting

### Essential Prior Knowledge

- **Basic algebra:** Manipulating equations, factoring, expanding brackets
- **GCSE/A-Level arithmetic:** Fractions, indices, surds
- **Basic graph sketching:** Understanding of x-y coordinate plane
- **Elementary logic:** Understanding of "if...then", "and", "or", "not"
- **Index laws:** Rules like a^m × a^n = a^(m+n)

### Helpful But Not Required

- A-Level Further Maths (provides head start on proof and series)
- Any prior exposure to set notation
- Basic programming experience (helps with understanding functions and logic)

### Prior Modules That Help

This is typically a first-year, first-semester module, so no university-level prerequisites. However, strong A-Level Mathematics (or equivalent) is assumed.

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **Simplify:** (x^3 × x^4) / x^2 = ?
   - *Expected answer: x^5*

2. **Solve:** 2x + 5 = 3x - 1. What is x?
   - *Expected answer: x = 6*

3. **Evaluate:** What is 5! (5 factorial)?
   - *Expected answer: 120*

4. **Logic:** If "All cats are animals" is true, is "All animals are cats" necessarily true?
   - *Expected answer: No*

5. **Expand:** (a + b)^2 = ?
   - *Expected answer: a^2 + 2ab + b^2*

**Scoring:** If you got 4-5 correct, you are well prepared. If fewer than 3, spend time revising basic algebra and arithmetic before lectures begin.
