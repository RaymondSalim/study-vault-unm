## Interactive Tools

Experiment with the core image processing algorithms below. All computations happen in your browser — try different inputs to build intuition.

**Available demos:**
- **Convolution Calculator** — enter a 3×3 kernel and image patch, see step-by-step multiplication and summation
- **Histogram Equalisation** — input pixel values, observe CDF mapping and before/after histograms
- **Huffman Coding** — type text to visualise frequency-based variable-length encoding
