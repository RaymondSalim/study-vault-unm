## Prerequisites

| Area | Specifics |
|------|-----------|

| Networking fundamentals | TCP/IP stack, HTTP, DNS, ports and protocols |

### Self-Test: Are You Ready?

Answer these 5 questions. If you struggle with more than 2, revise the prerequisite material first.

1. **What is `17 mod 5`?** (Expected: 2)
2. **Explain the difference between TCP and UDP in one sentence.** (Expected: TCP is connection-oriented with guaranteed delivery; UDP is connectionless with no delivery guarantee)
3. **What does HTTP status code 403 mean?** (Expected: Forbidden -- the server understood the request but refuses to authorise it)
4. **What is a stack frame and what does it typically contain?** (Expected: A section of the call stack holding a function's local variables, parameters, and return address)
5. **Write a basic SQL SELECT query that retrieves all users with age > 18.** (Expected: `SELECT * FROM users WHERE age > 18;`)
