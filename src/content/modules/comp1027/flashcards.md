## Key Concept Flashcards

| # | Question | Answer |
|---|----------|--------|
| 1 | What is a logic gate? | An electronic circuit that performs a Boolean operation on one or more binary inputs to produce a single output. |
| 2 | What are the three basic logic gates? | AND, OR, and NOT — all other gates can be constructed from combinations of these. |
| 3 | What is a NAND gate and why is it universal? | A NAND gate outputs 0 only when all inputs are 1; it is universal because any Boolean function can be built using only NAND gates. |
| 4 | What is a truth table? | A table listing all possible input combinations and their corresponding outputs for a logic circuit. |
| 5 | What is a combinational circuit? | A circuit whose output depends only on the current inputs (no memory), e.g., adders, multiplexers. |
| 6 | What is a sequential circuit? | A circuit whose output depends on current inputs and previous state (has memory), e.g., flip-flops, registers. |
| 7 | What is a flip-flop? | A sequential circuit element that stores one bit of information and changes state based on clock signals. |
| 8 | What is the difference between SR and D flip-flops? | SR has Set/Reset inputs (invalid state possible); D flip-flop stores whatever value is on the D input at the clock edge. |
| 9 | What is the Von Neumann architecture? | A computer architecture with a single memory for both instructions and data, a CPU with ALU and control unit, and input/output. |
| 10 | What are the components of a CPU? | ALU (Arithmetic Logic Unit), Control Unit, Registers, and connections to memory via buses. |
| 11 | What is the fetch-decode-execute cycle? | The CPU fetches an instruction from memory, decodes it to determine the operation, then executes it. |
| 12 | What is machine code? | Binary instructions directly executed by the CPU; each instruction specifies an operation and operands. |
| 13 | What is assembly language? | A low-level language using mnemonics (e.g., MOV, ADD) that map directly to machine code instructions. |
| 14 | What is the program counter (PC)? | A register that holds the memory address of the next instruction to be fetched. |
| 15 | What is the difference between RAM and ROM? | RAM is volatile read/write memory (lost on power off); ROM is non-volatile read-only memory (persists). |
| 16 | What is an operating system? | Software that manages hardware resources and provides services to application programs (process management, memory management, file systems). |
| 17 | What is a process vs a thread? | A process is an independent program in execution with its own memory; a thread is a lightweight unit of execution within a process sharing memory. |
| 18 | What is virtual memory? | A technique that uses disk space to extend available RAM, allowing programs to use more memory than physically available. |
| 19 | What is the OSI model? | A 7-layer networking reference model: Physical, Data Link, Network, Transport, Session, Presentation, Application. |
| 20 | What is the difference between TCP and UDP? | TCP is connection-oriented and reliable (guarantees delivery); UDP is connectionless and faster but unreliable. |
| 21 | What is an IP address? | A numerical label assigned to each device on a network, used for identification and routing (IPv4: 32-bit, IPv6: 128-bit). |
| 22 | What is a Karnaugh map used for? | Simplifying Boolean expressions by grouping adjacent 1s in a grid to find minimal sum-of-products form. |
| 23 | What is a multiplexer? | A combinational circuit that selects one of many input signals and forwards it to a single output based on select lines. |
| 24 | What is pipelining? | A CPU technique where multiple instructions are overlapped in execution, with different stages processed simultaneously. |
| 25 | What is a bus in computer architecture? | A communication system that transfers data between components; types include data bus, address bus, and control bus. |
