## Time Allocation

- **Total exam time:** Typically 2 hours
- **Reading time:** First 5 minutes — identify circuit-drawing vs theory questions
- **Circuit/calculation questions:** 10-15 minutes each (Boolean simplification, truth tables)
- **Short answer/definition:** 3-5 minutes each
- **Longer explanation questions:** 10-15 minutes each
- **Buffer:** 10 minutes for checking truth tables and circuit diagrams

## Question Types to Expect

- **Circuit drawing:** Design a circuit from a Boolean expression or specification
- **Truth table construction:** Fill in truth tables for given circuits
- **Boolean simplification:** Use algebra or Karnaugh maps to minimise expressions
- **Trace execution:** Step through machine code/assembly showing register values
- **Short answer:** Explain architectural concepts, compare protocols
- **Diagram labelling:** Label CPU components, OSI layers, network topologies

## Key Formulas/Facts to Memorise

1. Boolean algebra laws: De Morgan's, distributive, absorption, consensus
2. Karnaugh map grouping rules (powers of 2, wrap around)
3. Half adder: Sum = A XOR B, Carry = A AND B
4. Full adder: Sum = A XOR B XOR Cin, Cout = (A AND B) OR (Cin AND (A XOR B))
5. Fetch-decode-execute cycle steps
6. Von Neumann vs Harvard architecture differences
7. OSI model layers (mnemonic: "Please Do Not Throw Sausage Pizza Away")
8. TCP three-way handshake: SYN, SYN-ACK, ACK
9. Number conversions: binary ↔ decimal ↔ hexadecimal
10. Two's complement for negative numbers

## Night Before Checklist

1. ☐ Can you convert between binary, decimal, and hexadecimal quickly?
2. ☐ Can you simplify a Boolean expression using algebraic laws?
3. ☐ Can you draw and use a Karnaugh map for up to 4 variables?
4. ☐ Do you know the difference between combinational and sequential circuits?
5. ☐ Can you trace through the fetch-decode-execute cycle?
6. ☐ Can you explain what each CPU component does (ALU, CU, registers, PC)?
7. ☐ Can you trace through simple assembly/machine code instructions?
8. ☐ Do you know the OSI layers and what each one does?
9. ☐ Can you explain process scheduling and virtual memory at a high level?
10. ☐ Can you draw a half adder and full adder circuit?
