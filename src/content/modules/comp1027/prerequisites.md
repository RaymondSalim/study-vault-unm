## What You Need to Know Before Starting

### Essential Prior Knowledge

- **Binary number system:** Understanding of base-2 representation and conversion to/from decimal
- **Basic algebra:** Substitution and simplification of expressions
- **Logical thinking:** Comfortable with AND, OR, NOT concepts at an intuitive level
- **Basic arithmetic:** Addition, subtraction, multiplication in different bases

### Helpful But Not Required

- GCSE or A-Level Physics (electronics modules)
- Any prior exposure to digital logic or circuit simulation
- Basic understanding of how computers work at a high level
- Familiarity with hexadecimal notation

### Prior Modules That Help

This is typically a first-year module with no formal prerequisites. However:
- **COMP1017 - Mathematics for Computer Scientists 1:** Logic and set theory sections provide formal Boolean algebra background
- A-Level Computer Science (if taken) covers some architecture basics

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **Binary conversion:** What is 13 in binary?
   - *Expected answer: 1101*

2. **Basic logic:** If A = true and B = false, what is A AND B?
   - *Expected answer: false*

3. **Hexadecimal:** What is 0xFF in decimal?
   - *Expected answer: 255*

4. **Binary addition:** What is 1011 + 0110 in binary?
   - *Expected answer: 10001*

5. **General knowledge:** Name the three main components inside a CPU.
   - *Expected answer: ALU, Control Unit, Registers (or similar)*

**Scoring:** If you got 4-5 correct, you are well prepared. If fewer than 3, practise binary/hexadecimal conversions and review basic logic concepts before the module begins.
