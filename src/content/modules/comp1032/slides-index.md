# Slides Index

| Lecture | Topic | Key Concepts | Exam Weight |
|---------|-------|--------------|-------------|
| 1-2 | Introduction & Agents | Agent types (reflex, model-based, goal-based, utility-based), PEAS, environment properties | Low-Medium |
| 3-4 | Uninformed Search | BFS, DFS, iterative deepening, uniform cost search, completeness/optimality analysis | High |
| 5-6 | Informed Search | Heuristics, greedy best-first, A*, admissibility, consistency, IDA* | High |

## Tips for Using Slides

- Lectures 3-6 (search) form the core examinable content -- practise tracing algorithms on paper graphs
- Game playing (9-10) is a favourite exam topic; always practise alpha-beta on tree diagrams
- For ML, focus on understanding concepts and being able to explain trade-offs rather than complex maths
