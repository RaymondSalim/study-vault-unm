# Prerequisites

## What You Need Before This Module

- **Graph/tree concepts** -- understanding of nodes, edges, paths, and tree traversal
- **Basic logic** -- AND, OR, NOT, implication, truth tables
- **Algorithm thinking** -- ability to trace through step-by-step procedures
- **Basic probability** -- understanding of probability values, conditional probability (for ML section)
- **No advanced maths required** -- the module focuses on concepts and algorithm tracing rather than heavy calculation

## Prior Modules That Help

| Module | How it helps |
|--------|-------------|
| Any intro programming module | Algorithmic thinking, data structures (queues, stacks, trees) |
| Discrete maths / logic | Propositional logic, proof techniques, graph theory |
| Basic probability/statistics | Needed for machine learning section |

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **What data structure does BFS use?** (Expected: Queue / FIFO)
2. **If P = true and Q = false, what is P AND (NOT Q)?** (Expected: true)
3. **In a tree with branching factor 3 and depth 2, how many leaf nodes are there?** (Expected: 9)
4. **What is the difference between a stack and a queue?** (Expected: Stack is LIFO, queue is FIFO)
5. **If P(A) = 0.3 and P(B|A) = 0.5, what is P(A and B)?** (Expected: 0.15)

If you can answer at least 3 correctly, you are well-prepared. If not, review basic data structures, logic, and probability before the module begins.
