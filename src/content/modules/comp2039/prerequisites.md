# Prerequisites

## Required Background Knowledge

| Area | Topics Needed | Why |
|------|---------------|-----|
| Basic Programming | Loops, conditionals, arrays, functions | Implementing search algorithms and EAs |
| Probability & Statistics | Basic probability, expected values, random sampling | SA acceptance criteria, selection operators |
| Discrete Mathematics | Sets, permutations, combinations, graphs | Solution representations, search spaces |
| Basic Optimisation Concepts | Maxima/minima, objective functions | Understanding fitness landscapes |

## Helpful Prior Modules

- **COMP1013 - Algorithms & Data Structures** -- Understanding algorithmic complexity and graph traversal
- **COMP1011 - Computer Fundamentals** -- Binary representations (useful for GA encodings)
- **MATH1xxx - Probability/Statistics** -- Random variables, expected values, probability distributions
- **Any Programming module** -- Comfort with coding algorithms from pseudocode

## Key Concepts You Should Already Know

1. How to traverse a graph (BFS/DFS)
2. What an objective function is (something to maximise or minimise)
3. Basic probability calculations (P(A and B), P(A or B), conditional probability)
4. Binary encoding and decoding
5. Time complexity basics (why O(n^2) is worse than O(n log n))

## Self-Assessment Test

Answer these before starting the module. If you struggle with more than 2, review the prerequisites first.

| # | Question | Expected Answer |
|---|----------|-----------------|
| 1 | Given f(x) = -x^2 + 4x, what value of x maximises f(x)? | x = 2 (f(2) = 4) |
| 2 | If you flip a biased coin with P(H) = 0.7, what is P(two heads in a row)? | 0.7 x 0.7 = 0.49 |
| 3 | Convert the binary string 10110 to decimal. | 22 |
| 4 | In a graph with 5 nodes, what is the maximum number of edges (undirected, no self-loops)? | 5x4/2 = 10 |
| 5 | What does "greedy" mean in the context of algorithms? | Always choosing the locally optimal option at each step without backtracking |
