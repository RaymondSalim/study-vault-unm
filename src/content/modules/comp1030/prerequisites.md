# Prerequisites

## What You Need Before This Module

- **Basic arithmetic** -- addition, subtraction, multiplication, division
- **Exponents and powers of 2** -- comfortable with $2^0, 2^1, \ldots, 2^{15}$
- **Logical thinking** -- understanding of AND, OR, NOT at a conceptual level
- **No prior programming required** (assembly is taught from scratch)

## Prior Modules That Help

| Module | How it helps |
|--------|-------------|
| GCSE/A-Level Maths | Powers, arithmetic, basic algebra |
| A-Level Computer Science (if taken) | Binary, logic gates intro (not required) |

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **Convert decimal 25 to binary.** (Expected: 11001)
2. **What is $2^8$?** (Expected: 256)
3. **If A = true and B = false, what is A AND B?** (Expected: false)
4. **What is the decimal value of hexadecimal 0xF?** (Expected: 15)
5. **What is 1101 + 0011 in binary?** (Expected: 10000)

If you can answer at least 3 of these correctly, you have sufficient background. If not, review basic binary arithmetic and powers of 2 before lectures begin.
