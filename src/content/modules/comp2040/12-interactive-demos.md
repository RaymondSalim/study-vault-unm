## Interactive Tools

Build and simulate finite automata directly in your browser. Define states, transitions, and test input strings.

**Available demos:**
- **DFA/NFA Simulator** — configure states, alphabet, and transition function. Step through input strings one symbol at a time or run to completion. Includes presets for common languages.
