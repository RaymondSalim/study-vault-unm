# Flashcards

| # | Question | Answer |
|---|----------|--------|
| 1 | What is the difference between a DFA and an NFA? | A DFA has exactly one transition per symbol per state and no epsilon-transitions; an NFA can have multiple transitions per symbol, epsilon-transitions, or missing transitions. |
| 2 | What does the Pumping Lemma for regular languages prove? | It is used to prove a language is NOT regular by showing that sufficiently long strings cannot be "pumped" while remaining in the language. |
| 3 | What is the Chomsky hierarchy (top to bottom)? | Type 0: Recursively enumerable (Turing machine), Type 1: Context-sensitive, Type 2: Context-free (PDA), Type 3: Regular (DFA/NFA). |
| 4 | Define a regular expression. | A notation for describing regular languages using concatenation, union (|), and Kleene star (*) operations over an alphabet. |
| 5 | What is the closure property of regular languages under union? | If L1 and L2 are regular, then L1 union L2 is also regular. |
| 6 | How do you convert an NFA to a DFA? | Using the subset construction: each DFA state represents a set of NFA states, transitions are computed by taking the union of NFA transitions from all states in the set. |
| 7 | What is a pushdown automaton (PDA)? | A finite automaton augmented with a stack, capable of recognising context-free languages. |
| 8 | What is the relationship between CFGs and PDAs? | They are equivalent in power -- every context-free language can be recognised by a PDA and generated by a CFG. |
| 9 | State the Pumping Lemma for regular languages. | For any regular language L, there exists p >= 1 such that any string w in L with |w| >= p can be split as w = xyz where |xy| <= p, |y| >= 1, and xy^i z is in L for all i >= 0. |
| 10 | What is a context-free grammar (CFG)? | A grammar with production rules of the form A -> alpha, where A is a single non-terminal and alpha is a string of terminals and non-terminals. |
| 11 | What makes a language decidable? | A language is decidable if there exists a Turing machine that accepts every string in the language and rejects every string not in it (always halts). |
| 12 | What is the halting problem? | The problem of determining whether a given Turing machine halts on a given input; it is undecidable. |
| 13 | Define an alphabet, string, and language. | Alphabet: finite set of symbols. String: finite sequence of symbols from an alphabet. Language: set of strings over an alphabet. |
| 14 | What is Kleene star (sigma*)? | The set of all possible finite strings over alphabet sigma, including the empty string epsilon. |
| 15 | What is the difference between a recogniser and a decider? | A recogniser accepts strings in the language (may loop on non-members); a decider always halts with accept or reject for any input. |
| 16 | How does a Turing machine differ from a PDA? | A TM has an infinite read/write tape and can move left or right; a PDA has a stack (LIFO only) and a read-only input tape. TMs are strictly more powerful. |
| 17 | What is ambiguity in a CFG? | A CFG is ambiguous if there exists at least one string with two or more distinct leftmost derivations (or parse trees). |
| 18 | What is the language accepted by the empty DFA (no accepting states)? | The empty language (no strings are accepted). |
| 19 | Explain the subset construction complexity. | An NFA with n states can produce a DFA with up to 2^n states in the worst case. |
| 20 | What is epsilon-closure? | The set of all states reachable from a given state using only epsilon-transitions (including the state itself). |
| 21 | Can context-free languages be closed under intersection? | No, CFLs are not closed under intersection (or complement). |
| 22 | What is a derivation in a CFG? | A sequence of production rule applications starting from the start symbol, ultimately producing a string of terminals. |
| 23 | What does it mean for a Turing machine to be universal? | A universal TM can simulate any other TM given its description and input -- it is a programmable computer. |
| 24 | What is the Church-Turing thesis? | The thesis states that any effectively computable function can be computed by a Turing machine; TMs capture the informal notion of "algorithm." |
| 25 | What is the Myhill-Nerode theorem used for? | It characterises regular languages via equivalence classes and can be used to find the minimum-state DFA for a language. |
