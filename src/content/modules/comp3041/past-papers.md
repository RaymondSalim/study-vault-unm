## Practice Questions

Since no past papers are available, use these practice questions for revision:

### Scenario Analysis (Long-form)

1. **A social media company discovers its recommendation algorithm amplifies misinformation but also maximises user engagement (and therefore revenue). Analyse this dilemma using consequentialism, deontology, and virtue ethics. What should the company do?**

2. **A developer discovers their employer's software collects user location data beyond what the privacy policy states. Apply the SE Code of Ethics and Birsch's accountability criteria to determine what the developer should do.**

3. **A health insurance company purchases fitness app data to adjust premiums. Analyse using Nissenbaum's Contextual Integrity framework — identify context, actors, norms, and whether CI is violated.**

### Short Answer

4. Identify the fallacy: "We should ignore Professor Chen's argument about AI safety because she has investments in competing companies."
5. Explain the difference between GPL and MIT licences using one concrete example.
6. Name the four requirements for fair use and apply them to: a student sharing 3 chapters of a textbook on a class WhatsApp group.
7. Explain the Problem of Many Hands using the Boeing 737 MAX as an example.

### Definitions

8. Define: accountability, responsibility, liability. Give a computing example for each.
9. What are the three types of research misconduct? Give an example of each.
10. List the 8 principles of the SE Code of Ethics.
