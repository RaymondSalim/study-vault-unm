# Slides Index

| Lecture | Topic | Key Concepts | Exam Weight |
|---------|-------|--------------|-------------|
| 1-2 | Vectors & Matrices | Vector operations, dot/cross product, matrix multiplication, transpose, inverse | Medium |
| 3-4 | Linear Systems | Gaussian elimination, row echelon form, solution types (unique, infinite, none) | Medium |
| 5-6 | Determinants & Eigenvalues | Determinant computation, properties, eigenvalue equation $Av = \lambda v$, characteristic polynomial | Medium-High |
| 7-8 | Linear Transformations | Rotation, reflection, scaling matrices, composition of transformations, kernel/image | Medium |
| 9-11 | Calculus | Differentiation (chain, product, quotient rules), integration (by parts, substitution, partial fractions), partial derivatives | High |
| 12-13 | Probability | Sample spaces, conditional probability, Bayes' theorem, random variables, expectation, variance | High |
| 14-15 | Distributions | Binomial, Poisson, Normal, standardisation, CLT, hypothesis testing intro | High |
| 16 | Graph Theory | Graphs, trees, paths, Euler/Hamilton circuits, adjacency matrices, isomorphism | Low-Medium |

## Tips for Using Slides

- Calculus (lectures 9-11) and probability/distributions (12-15) together make up the majority of exam marks -- prioritise these
- Linear algebra (1-8) forms a coherent block; make sure to understand matrix operations before eigenvalues
- Graph theory (16) is typically a smaller exam component but straightforward marks if prepared
- Practise computation by hand -- exams require showing working, not just final answers
