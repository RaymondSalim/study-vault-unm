## What You Need to Know Before Starting

### Essential Prior Knowledge

- **Basic computer literacy:** Comfortable using a computer, file system navigation, text editors
- **Logical thinking:** Ability to follow step-by-step instructions and reason about conditions
- **Basic mathematics:** Understanding of variables, expressions, and simple algebra
- **Problem decomposition:** Breaking a task into smaller, manageable steps

### Helpful But Not Required

- Prior programming experience in any language (Python, JavaScript, Scratch)
- A-Level Computer Science (provides a significant head start)
- Familiarity with any IDE (IntelliJ, VS Code, Eclipse)
- Understanding of binary number representation

### Prior Modules That Help

This is typically a first-year, first-semester module with no formal university prerequisites. However:
- **COMP1017 - Mathematics for Computer Scientists 1:** Logical thinking and formal reasoning skills transfer well
- Any A-Level or GCSE Computing course

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **Logic:** If x = 5 and the condition is "x > 3 AND x < 10", is it true or false?
   - *Expected answer: true (both conditions are satisfied)*

2. **Sequencing:** What is the result of: start with 0, add 3, multiply by 2, subtract 1?
   - *Expected answer: 5*

3. **Repetition:** If you repeat "add 2" starting from 0, five times, what do you get?
   - *Expected answer: 10*

4. **Problem solving:** To find the largest number in a list, what strategy would you use?
   - *Expected answer: Go through each number, keeping track of the biggest one seen so far*

5. **Abstraction:** If you have instructions to make tea, what is the "input" and "output"?
   - *Expected answer: Input: water, tea bag, cup; Output: a cup of tea*

**Scoring:** If you got 4-5 correct, you are well prepared. If fewer than 3, practise logical reasoning and step-by-step problem solving before the module begins. No prior programming knowledge is strictly required.
