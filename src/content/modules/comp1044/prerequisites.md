# Prerequisites

## What You Need Before This Module

- **Set theory basics** -- understanding of sets, subsets, unions, intersections (for relational algebra)
- **Logical operators** -- AND, OR, NOT for query conditions
- **Basic HTML/web concepts** -- helpful for the web interfaces section
- **No prior SQL or database experience required** -- taught from scratch

## Prior Modules That Help

| Module | How it helps |
|--------|-------------|
| Any intro programming module | Logical thinking, understanding data types |
| Discrete maths | Set theory, relations, functions -- directly maps to relational model |
| Web development (if taken) | HTML/CSS/JS knowledge speeds up the interfaces section |

## Quick Self-Test

Answer these 5 questions to check your readiness:

1. **What is a set?** (Expected: An unordered collection of distinct elements)
2. **If A = {1,2,3} and B = {2,3,4}, what is A intersect B?** (Expected: {2,3})
3. **What does the logical expression (age > 18 AND status = 'active') mean?** (Expected: Both conditions must be true simultaneously)
4. **What is a table in everyday terms?** (Expected: A structure with rows and columns organising related data)
5. **What is HTML used for?** (Expected: Structuring content on web pages)

If you can answer at least 3 correctly, you have sufficient background. If not, review basic set theory and logical operators before starting.
